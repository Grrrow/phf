# Editorial Text Section

## Descripción General
Este es el bloque de texto principal para escribir la biografía. Está diseñado para ser muy legible, controlando automáticamente la longitud máxima de la línea de texto para que los ojos no se cansen.

## Cómo Usarlo
- **Text**: Aquí es donde escribes la biografía. Utiliza el editor "RichText" para hacer negritas, cursivas o añadir enlaces internos.
- **Title y Eyebrow**: Puedes añadir un título a la sección de texto (ej. "The Architect of Sound").
- **Text Width**: Por defecto, el ancho de lectura está en `Medium`, que es el óptimo para leer largos párrafos. Puedes cambiarlo a `Narrow` o `Wide` si el diseño lo requiere.
- **Alignment**: Puedes decidir si los textos están alineados a la izquierda (como un libro) o centrados.
- **Spacing**: Tienes controles para separar este bloque del anterior o del posterior. Muy útil si tienes varios bloques de texto seguidos y quieres unirlos visualmente poniendo un `spacing_top: none`.
