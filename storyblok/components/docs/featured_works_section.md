# Featured Works Section

## Descripción General
La `Featured Works Section` es un bloque diseñado para destacar composiciones, proyectos y álbumes de manera editorial, muy elegante y asimétrica. Se usa principalmente para reforzar la faceta de compositor.

## Uso y Ubicación
Puedes insertar este bloque en el campo `body` de la Home o cualquier página (ej. Works, About, Discography).

## Variantes de Diseño (Layout Variants)
- **Editorial Grid (Recomendado)**: Crea un layout asimétrico de hasta 3 columnas en pantallas grandes. Requiere que combines diferentes tamaños de tarjeta (`card_size`) en los ítems interiores para que el efecto visual se construya correctamente.
- **Compact Grid**: Layout tradicional de 2 columnas.
- **Masonry**: Las tarjetas se apilan como un muro de ladrillos, rellenando huecos verticales.

## Uso del CTA (Call To Action)
Si deseas enlazar hacia la agenda completa u otra página, rellena los campos `cta_label` y `cta_link`. Aparecerá un botón centrado bajo todas las tarjetas.

## Buenas Prácticas
- Asegúrate de rellenar los campos `eyebrow`, `title` y `subtitle` para contextualizar la selección.
- Deja `show_decorative_line` en `true` si usas un tema claro, aporta un toque estético sutil.
- Combina al menos una tarjeta tamaño `Large` (con imagen) y varias tarjetas tamaño `Medium`/`Small` para crear el efecto editorial deseado.
