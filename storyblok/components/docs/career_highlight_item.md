# Career Highlight Item

## Descripción General
Este bloque representa una sola fila dentro de la sección "Career Highlights". Visualmente es muy sobrio: muestra la fecha a la izquierda, y a la derecha (o debajo en móviles) el cargo y la descripción.

## Rellenar la Fila
- **Date Label**: No tienes que forzarte a usar un formato de fecha estricto. Escribe exactamente lo que quieres que lea el usuario: `2001 - 2004`, `Desde 2018`, o `Actualidad`.
- **Role Label**: El cargo que ocupó (ej. `Artistic Director`, `Principal Guest Conductor`). Se mostrará pequeño, en mayúsculas y en color dorado/acento para diferenciar jerarquías visuales.
- **Description**: El texto gris descriptivo detallando lo que logró o hizo en esa etapa. Intenta mantenerlo en un solo párrafo para que el listado visual no se vuelva demasiado denso.
- **Sort Order**: Un campo técnico oculto. No le prestes atención a menos que un desarrollador te indique que es necesario para reordenar cosas por código.
