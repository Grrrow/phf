# Works Catalog Page

## Descripción General
Página especializada para mostrar un catálogo editorial completo de las obras del compositor. Tiene un diseño de listas asimétricas, con tipografía clásica, mucho aire y un estilo marcadamente elegante.

## Cómo Crear la Página en Storyblok
1. En Storyblok, crea una nueva Story (página).
2. Como **Content Type**, selecciona `works_catalog_page`.
3. Nombra la página (recomendado: "Works" u "Obras").
4. El Slug recomendado es `/works/` o `/obras/` (para EN y ES respectivamente).

## Estructura de Contenido
- **SEO**: Añade el bloque de SEO para configurar el title y meta description que leerá Google.
- **Title e Intro**: Serán el H1 y el párrafo introductorio de la página. Rellénalos en español y cambia a inglés en el selector superior para la traducción.
- **Groups**: Aquí dentro es donde debes ir añadiendo los bloques `Works Catalog Group` (por ejemplo, un grupo para 'Operas', otro para 'Symphonic', etc.).

## Temas de la Página (Page Theme)
Puedes elegir el ambiente general de la página:
- **Editorial (Off-white)**: (Recomendado). Un fondo gris muy claro cálido, ideal para catálogos.
- **Light**: Fondo totalmente blanco.
- **Dark**: Fondo oscuro para una presencia más dramática.

## Idiomas
Todos los campos visibles (`title`, `intro`) son completamente traducibles. No debes crear dos páginas separadas para los idiomas, la arquitectura actual maneja las traducciones a nivel de campo (Field-level translation).
