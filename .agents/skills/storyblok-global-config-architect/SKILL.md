# Storyblok Global Config Architect Skill

This skill focuses on designing global, structural blocks and configurations.

## Instructions
When tasked with designing global configurations (especially for cultural websites), you MUST:
1. **Design Global Blocks:** Design blocks that represent global concepts, such as `global_config`, `header_config`, `footer_config`, and `seo`.
2. **Multi-language Support:** Design schemas with internationalization (multi-language) in mind.
3. **SEO Optimization:** Include necessary SEO fields (meta titles, descriptions, open graph images).
4. **Social & Navigation:** Incorporate fields for social media links, navigation menus, footer links, and global settings.
5. **Component Types:** Recommend and apply the appropriate Storyblok component type:
   - `root`: For content types/pages.
   - `nestable`: For regular blocks.
   - `singleton` or `global`: For single-instance configuration blocks.
